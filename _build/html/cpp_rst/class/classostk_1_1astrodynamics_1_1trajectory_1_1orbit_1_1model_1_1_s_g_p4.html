    Class ostk::astrodynamics::trajectory::orbit::model::SGP4¶

    class SGP4 : public ostk::astrodynamics::trajectory::orbit::Model¶

    SGP4 orbit model.

    Simplified General Perturbations 4 (SGP4) orbit propagation model using Two-Line Element (TLE) sets.

    Public Functions

    SGP4(const TLE &aTle)¶

    Construct an SGP4 model from a TLE. 

    TLE tle = TLE("1 25544U 98067A   08264.51782528 -.00002182  00000-0 -11606-4 0  2927",
              "2 25544  51.6416 247.4627 0006703 130.5360 325.0288 15.72125391563537");
SGP4 sgp4 = SGP4(tle);
    Parameters:

    aTle -- A Two-Line Element set.

    SGP4(const TLE &aTle, const Shared<const Frame> &anOutputFrameSPtr)¶

    Construct an SGP4 model from a TLE and an output frame. If TEME, the runtime is faster as no frame transformations are needed. In other frames, the runtime will be slower as frame transformations are needed. 

    TLE tle = TLE("1 25544U 98067A   08264.51782528 -.00002182  00000-0 -11606-4 0  2927",
              "2 25544  51.6416 247.4627 0006703 130.5360 325.0288 15.72125391563537");
SGP4 sgp4 = SGP4(tle, Frame::TEME());
    Parameters:

    • aTle -- A Two-Line Element set.

    • anOutputFrameSPtr -- An output frame.
