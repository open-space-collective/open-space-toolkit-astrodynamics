    class Profile¶

    Spacecraft flight profile.

    Public Types

    enum class PointingMode¶

    Values:

    enumerator Undefined¶

    Undefined pointing mode.

    enumerator Inertial¶

    Inertial pointing mode (the spacecraft points to a celestial object)

    enumerator Nadir¶

    Nadir pointing mode (the spacecraft points points "directly down")

    enumerator Target¶

    Target pointing mode (the spacecraft points to a given target position)

    enumerator Custom¶

    Custom pointing mode.

    enum class Axis¶

    Values:

    enumerator X¶
    enumerator Y¶
    enumerator Z¶
    enum class TargetType¶

    Values:

    enumerator GeocentricNadir¶
    enumerator GeodeticNadir¶

    Negative of the position vector of the satellite in the ECI frame.

    enumerator Trajectory¶

    Negative of the geodetic normal of the satellite in the ECI frame.

    enumerator TargetPosition¶

    DEPRECATED: Use TargetPosition instead.

    enumerator TargetVelocity¶

    Points towards the provided target position.

    enumerator TargetSlidingGroundVelocity¶

    Points along the provided target's velocity vector.

    enumerator Sun¶

    Points along the provided target's ground velocity vector (aka the scan direction of the point sliding across the ground)

    enumerator Moon¶

    The position of the Sun.

    enumerator VelocityECI¶

    The position of the Moon.

    enumerator VelocityECEF¶

    The velocity vector in the ECI frame.

    enumerator OrbitalMomentum¶

    The velocity vector in the ECEF frame.

    enumerator OrientationProfile¶

    The orbital momentum vector of the satellite in the ECI frame.

    enumerator Custom¶

    Points towards a profile of orientations in the ECI frame.

    • isAntiDirection -- Whether the target is in the anti-direction.

    static TrajectoryTarget TargetSlidingGroundVelocity(const ostk::astrodynamics::Trajectory &aTrajectory, const Axis &anAxis, const bool &isAntiDirection = false)¶

    Constructs a TrajectoryTarget object of type TargetSlidingGroundVelocity, pointing along the ground velocity vector (aka the scan direction of the point sliding across the ground). This will compensate for the rotation of the referenced celestial body.

    Parameters:

    • anAxis -- The axis of the target.

    • isAntiDirection -- Whether the target is in the anti-direction.
