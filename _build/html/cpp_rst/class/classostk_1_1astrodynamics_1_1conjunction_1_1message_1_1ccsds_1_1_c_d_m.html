    Public Types

    enum class ObjectType¶

    Values:

    enumerator Payload¶

    Operational payload spacecraft.

    enumerator RocketBody¶

    Rocket body or upper stage.

    enumerator Debris¶

    Orbital debris fragment.

    enumerator Unknown¶

    Object type is unknown.

    enumerator Other¶

    Object type not covered by other values.
