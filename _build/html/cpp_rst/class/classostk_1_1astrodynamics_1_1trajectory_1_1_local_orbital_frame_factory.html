    class LocalOrbitalFrameFactory¶

    A local orbital frame factory.

    Creates local orbital reference frames (e.g., LVLH, TNW, QSW, VNC, NED) from a given state (position and velocity). The factory generates frames on demand, parameterized by a parent frame and a frame type.
