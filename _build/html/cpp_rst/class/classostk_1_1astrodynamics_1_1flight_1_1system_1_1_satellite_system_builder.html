    Class ostk::astrodynamics::flight::system::SatelliteSystemBuilder¶

    class SatelliteSystemBuilder¶

    Satellite System builder, meant to simplify creation of a SatelliteSystem, by allowing you to only specify the parameters you want. There are two ways of doing this: 

    SatelliteSystemBuilder().withDryMass(X).withArea(Y)

SatelliteSystemBuilder::Default().withDryMass(X)
