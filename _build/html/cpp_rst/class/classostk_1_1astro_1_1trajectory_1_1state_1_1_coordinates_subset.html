    Class ostk::astro::trajectory::state::CoordinatesSubset¶

    class CoordinatesSubset¶

    State coordinates subset. It contains information related to a particular group of coordinates. It does not contain the coordinate values.

    Subclassed by ostk::astro::trajectory::state::coordinatessubsets::AngularVelocity, ostk::astro::trajectory::state::coordinatessubsets::AttitudeQuaternion, ostk::astro::trajectory::state::coordinatessubsets::CartesianPosition, ostk::astro::trajectory::state::coordinatessubsets::CartesianVelocity
