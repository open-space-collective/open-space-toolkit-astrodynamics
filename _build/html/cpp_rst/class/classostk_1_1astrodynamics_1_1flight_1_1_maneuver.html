    Maneuver toConstantLocalOrbitalFrameDirectionManeuver(const Shared<const LocalOrbitalFrameFactory> &aLocalOrbitalFrameFactorySPtr, const Angle &aMaximumAllowedAngularOffset = Angle::Undefined()) const¶

    Create a new maneuver with a constant thrust acceleration direction in the Local Orbital Frame.

    The new Maneuver contains the same states as the original Maneuver, but the thrust acceleration direction is constant in the Local Orbital Frame. Said direction is the mean direction of the thrust acceleration directions in the Local Orbital Frame of the original Maneuver. The thrust acceleration magnitude profile is the same as the original.

    If defined, a runtime error will be thrown if the maximum allowed angular offset between the original thrust acceleration direction and the mean thrust direction is violated.

    Maneuver newManeuver =
maneuver.toConstantLocalOrbitalFrameDirectionManeuver(LocalOrbitalFrameFactory::TNW(Frame::GCRF()));
    Parameters:

    • aLocalOrbitalFrameFactorySPtr -- A local orbital frame factory

    • aMaximumAllowedAngularOffset -- A maximum allowed angular offset to consider (if any). Defaults to Undefined.

    Returns:

    A new maneuver
