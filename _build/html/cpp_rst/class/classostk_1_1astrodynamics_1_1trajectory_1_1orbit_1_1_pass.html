    Class ostk::astrodynamics::trajectory::orbit::Pass¶

    class Pass¶

    A revolution of an orbiting object.

    This class represents a pass, which is a revolution of an orbiting object. It provides methods to get the type of the pass, the revolution number, and the instants at various points of the pass.

    See also

    http://help.agi.com/stk/11.3.0/index.htm#vo/sat_pass.htm
