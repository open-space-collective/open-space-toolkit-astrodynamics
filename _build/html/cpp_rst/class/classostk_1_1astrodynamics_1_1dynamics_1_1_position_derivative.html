    Class ostk::astrodynamics::dynamics::PositionDerivative¶

    class PositionDerivative : public ostk::astrodynamics::Dynamics¶

    Position derivative dynamics.

    Defines the contribution to the position state derivative due to velocity. This dynamics simply maps the velocity components of the state vector to the position derivative, implementing the kinematic relationship dx/dt = v.
