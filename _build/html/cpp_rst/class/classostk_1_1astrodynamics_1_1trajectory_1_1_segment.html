    enum class MaximumManeuverDurationViolationStrategy¶

    Maximum maneuver duration violation strategy Strategy to use when a maneuver exceeds the maximum duration constraint.

    Example: Proposed maneuver: [-----------------&#8212;|---------&#8212;] TruncateEnd: [-----------------&#8212;] TruncateStart: [-----------------&#8212;] Center: [-----------------&#8212;]

    Values:

    enumerator Fail¶

    Will throw a RuntimeError if a maneuver exceeds the maximum duration.

    enumerator Skip¶

    The maneuver will be skipped entirely.

    enumerator TruncateEnd¶

    The maneuver will be shortened to the maximum duration, truncating the end segment.

    enumerator TruncateStart¶

    The maneuver will be shortened to the maximum duration, truncating the start segment.

    enumerator Center¶

    The maneuver will be shortened to the maximum duration, truncating the edges, keeping the centered part of the maneuver.

    static Segment ConstantLocalOrbitalFrameDirectionManeuver(const String &aName, const Shared<EventCondition> &anEventConditionSPtr, const Shared<Thruster> &aThrusterDynamics, const Array<Shared<Dynamics>> &aDynamicsArray, const NumericalSolver &aNumericalSolver, const Shared<const LocalOrbitalFrameFactory> &aLocalOrbitalFrameFactory, const Angle &aMaximumAllowedAngularOffset = Angle::Undefined(), const ManeuverConstraints &aManeuverConstraints = ManeuverConstraints())¶

    Create a maneuvering segment that produces maneuvers with a constant direction in the local orbital frame.

    The provided thruster dynamics are used to solve the segment at first. The maneuvers produced by this segement solution are then used to create a new thruster dynamics with a constant direction in the local orbital frame. This new thruster dynamics is then used to actually solve the segment.

    If defined, a runtime error will be thrown if the maximum allowed angular offset between the original thruster dynamics and the mean thrust direction is violated.

    Parameters:

    • aName -- A name

    • anEventConditionSPtr -- An event condition

    • aThrusterDynamics -- Dynamics for the thruster

    • aDynamicsArray -- Array of dynamics

    • aNumericalSolver -- Numerical solver

    • aLocalOrbitalFrameFactory -- A local orbital frame factory.

    • aMaximumAllowedAngularOffset -- A maximum allowed angular offset to consider (if any). Defaults to Undefined.

    • aManeuverConstraints -- Maneuver constraints

    Returns:

    A Segment for maneuvering
