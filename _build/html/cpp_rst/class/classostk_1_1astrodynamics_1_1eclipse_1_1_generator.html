    class Generator¶

    Eclipse generator for trajectory analysis.

    Generates eclipses for a given trajectory over a specified analysis interval.

    Public Functions

    Generator(const Environment &anEnvironment = Environment::Default(), const Duration &aSearchStepSize = Duration::Seconds(60.0), const Duration &aSearchTolerance = Duration::Milliseconds(1.0))¶

    Constructor. 

    Generator eclipseGenerator = { environment, Duration::Seconds(60.0), Duration::Milliseconds(1.0) };
    Parameters:

    • anEnvironment -- The Environment to use during the search. Defaults to the default environment.

    • aSearchStepSize -- The step size to use during the search. Defaults to 60 seconds.

    • aSearchTolerance -- The tolerance to use during the search. Defaults to 1 millisecond.
