    class FiniteDifferenceSolver¶

    Finite Difference solver.

    Computes numerical derivatives (State Transition Matrices, gradients, Jacobians) using forward, backward, or central finite difference schemes.

    Public Types

    enum class Type¶

    Finite difference scheme type.

    Values:

    enumerator Forward¶

    Forward finite difference scheme.

    enumerator Backward¶

    Backward finite difference scheme.

    enumerator Central¶

    Central finite difference scheme.
