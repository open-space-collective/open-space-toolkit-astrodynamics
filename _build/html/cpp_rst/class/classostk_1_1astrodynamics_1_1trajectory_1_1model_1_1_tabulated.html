    Class ostk::astrodynamics::trajectory::model::Tabulated¶

    class Tabulated : public virtual ostk::astrodynamics::trajectory::Model¶

    Tabulated trajectory model. 

                     Interpolation is performed between states using the specified interpolation scheme.
                 For now, linear, barycentric rational and cubic spline interpolation schemes are
                 supported.

    Subclassed by ostk::astrodynamics::trajectory::orbit::model::Tabulated
