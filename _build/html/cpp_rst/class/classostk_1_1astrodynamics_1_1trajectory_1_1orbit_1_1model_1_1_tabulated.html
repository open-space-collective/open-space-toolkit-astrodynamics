    class Tabulated : public virtual ostk::astrodynamics::trajectory::orbit::Model, public ostk::astrodynamics::trajectory::model::Tabulated¶

    Tabulated orbit model.

    Orbit model based on tabulated states with interpolation, extending the trajectory tabulated model with revolution number tracking.
